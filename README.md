# Weatherful - Weather Dashboard

## Purpose

To create a weather dashboard displaying weather from today and the next 5 days.

## Built With

- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery
- Moment.js
- OpenWeatherMap API
